Critical Gotchas

FK Type Mismatch: project_id must be INTEGER (not UUID). projects.id is number in the database. This caused a production bug in scheduling (2026-01-28).
Next.js 15 Async Params: params must be awaited in API routes and pages. const { projectId } = await context.params;
Route Naming: Use [projectId], [drawingId], [areaId]. NEVER use generic [id].
.next Cache: After creating/modifying routes, run rm -rf .next before testing. Stale cache causes false 404s.
Storage RLS: Uploads fail silently without storage policies. Check 20260201000002_add_storage_rls_policies.sql.
Supabase Client Context: Use @/lib/supabase/server in API routes, @/lib/supabase/client in browser.
