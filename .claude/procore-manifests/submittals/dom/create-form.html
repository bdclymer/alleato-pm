Learn How to Add Submittals Using Procore Imports

Procore Imports is the most common method for adding submittals to Procore and the most efficient way to bulk generate a submittal registry without having to add each submittal manually.

Learn How to Generate Submittals from Specifications

Using Procore’s Submittal Builder is the quickest and most accurate way to create submittals. It scans specifications you’ve added to the Specifications tool and automatically generates a submittal register in minutes.

Learn How to Manually Create Submittals Items

Easily create individual submittals and send them for review. Procore maintains a history of every submittal, so you can track who created the submittal, when it was created, and the date it was received.
