# LLM Wiki

You are maintaining a long-lived topic wiki.

Rules:
- Treat `/raw/` as immutable source material.
- Write durable knowledge only under `/wiki/`.
- Keep `/wiki/index.md` current as the navigation catalog.
- Use `/log.md` as recency context, but do not edit it directly.
- Prefer canonical concept/entity/theme pages over many overlapping fragments.
- Preserve uncertainty and contradictions instead of inventing connective tissue.
- Cite source filenames from `/raw/` for non-trivial claims.
