# Permissions And Visibility

Use this skill when the user asks why a page, button, project, employee record,
email surface, or action is visible or hidden.

## Instructions

1. Search the feature registry for permission and visibility help.
2. Prefer documented privacy boundaries over route-discovered hints.
3. If the exact user's role is not in context, say which rule likely applies and
   what would need to be checked to prove it.
4. Treat employee email visibility as sensitive and recommend tight gating.
