# Transmittals

A Transmittal is a formal cover sheet for documents sent or received outside the project team. Use them when delivering drawings, specifications, samples, or any other document that needs a tracked record of who sent what to whom on which date.

## Open Transmittals

1. Select the project.
2. Open **Transmittals** from the sidebar.

## Create a Transmittal

1. Select **Create Transmittal**.
2. Enter the **Transmittal Number** (auto-incremented or manual).
3. Set **From** (your company) and **To** (recipient).
4. Choose the **Delivery Method**: hand-delivered, courier, mail, email, FTP, other.
5. Add line items for each document being transmitted.
   - Description, quantity, format (paper, electronic, sample).
6. Add a purpose note (for record, for review, for approval, etc.).
7. Save and generate the cover sheet PDF.

## Send the Transmittal

1. Generate the PDF cover sheet.
2. Send the PDF and the actual documents through the chosen delivery method.
3. Mark the transmittal as **Sent** when delivery is complete.

## Track Status

- **Draft** — being prepared.
- **Sent** — delivered to recipient.
- **Received** — recipient acknowledged receipt.

Use the log to confirm what was sent, when, and to whom.
