# Project Home

Project Home is the dashboard for a single project. It is the fastest way to get oriented before opening a specific tool.

## Open Project Home

1. Select the project in the header.
2. Project Home opens by default, or select **Home** from the sidebar.

## What It Shows

- **KPI strip** — schedule status, budget variance, open RFIs, pending submittals.
- **Team roster** — key project team members with roles and contact info.
- **Open RFIs** — items waiting on you or the design team.
- **Pending submittals** — submittals in review or awaiting submission.
- **Recent meetings** — last few meetings with extracted action items.
- **Budget summary** — original, revised, committed, actual, forecast.
- **Active commitments** — top subcontracts and POs by value.

## Use It For Triage

Project Home is built for the first five minutes of your day on a project. It surfaces what needs your attention so you do not have to open every tool to find it.
