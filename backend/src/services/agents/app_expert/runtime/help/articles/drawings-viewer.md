# Drawing Viewer and Annotations

The drawing viewer opens any drawing as a full-screen PDF with annotation, measurement, and pin tools. Markups save to the drawing record and stay visible to the project team.

## Open the Viewer

1. Open **Drawings**.
2. Select a drawing from the log.
3. The viewer opens to the current revision.

## Annotation Tools

The toolbar provides:

- **Pen** — freehand markup.
- **Highlighter** — translucent highlight.
- **Rectangle** — outlined box for callouts.
- **Arrow** — directional indicator.
- **Text** — typed annotation.
- **Eraser** — remove markup.
- **Comment** — text comment with thread replies.

Each annotation is timestamped and attributed to its author.

## Pin to RFIs, Punch Items, and Photos

Drop a pin on the drawing to link it to another record:

1. Select the **Pin** tool.
2. Click the location on the drawing.
3. Choose the link type: **RFI**, **Punch Item**, or **Photo**.
4. Select an existing record or create a new one inline.

The pin stays on the drawing. Opening the linked record shows the drawing pin location.

## Compare Revisions

When a drawing has multiple revisions:

1. Select **Compare** in the viewer.
2. Choose the revision to compare against.
3. Differences highlight in color.

## Print and Export

- **Print** — opens the browser print dialog with markups optionally included.
- **Export** — download the drawing with or without markups as a flattened PDF.
