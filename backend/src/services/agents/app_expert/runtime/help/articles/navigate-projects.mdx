# Navigate Projects

[Projects](/help/articles/navigate-projects) are the main workspace in Alleato OS. Most project tools use the
selected project context, so confirm the right project before reviewing or
changing project information.

## Switch Projects

1. Use **Select Project** in the header.
2. Search for the project by name or job number.
3. Select the project from the list.
4. Use [View all projects](/help/articles/navigate-projects) if you need to browse the full project
   list.

After you select a project, project-specific tools open in that project's
context.

## Open Project Home

Use the [project home page](/help/articles/navigate-projects) when you need the fastest orientation point. Project
home summarizes the project and gives you a starting place before moving into
budgets, documents, drawings, meetings, change management, or other tools.

## Check Before Acting

- Confirm the selected project name before uploading, editing, or approving
  anything.
- If a project does not appear, ask an admin to confirm your project access.
- If a tool appears empty, check that you are in the intended project before
  reporting missing data.
