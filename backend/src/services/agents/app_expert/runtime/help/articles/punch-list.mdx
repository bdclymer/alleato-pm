# Punch List

The Punch List tracks deficiencies and incomplete items found during pre-completion walks. Each item has an assignee, location, due date, and status so the project can close out cleanly.

## Open the Punch List

1. Select the project.
2. Open **Punch List** from the sidebar.

## Create a Punch Item

1. Select **Add Punch Item**.
2. Enter a **Description** of the deficiency.
3. Set the **Location** (room, area, floor, or drawing pin).
4. Assign a responsible party (subcontractor or trade).
5. Set a **Due Date**.
6. Set **Priority** (low, medium, high).
7. Attach photos showing the issue.
8. Save the item.

## Pin to a Drawing

Punch items can be pinned to a sheet in the [Drawings](/help/articles/drawings) viewer. Open the drawing, drop a pin, and link it to the punch item. The pin and item stay synced.

## Status Workflow

- **Open** — newly created, not addressed.
- **In Progress** — assignee is working on it.
- **Ready for Review** — assignee marked complete, awaiting verification.
- **Closed** — verified complete.
- **Void** — no longer applicable.

## Bulk Actions

Use the table view to filter by trade or assignee, then bulk-update status, due date, or assignee for multiple items at once.
