    ## Submittal Package
    A _submittal package_ is a container that stores one or more _submittals_. Typically, a _general contractor_ creates submittal packages that list all of the individual submittals specific to a particular trade or _subcontractor_. For example, one might create a submittal package to contain all of the plumbing-related submittals in a commercial building project.
