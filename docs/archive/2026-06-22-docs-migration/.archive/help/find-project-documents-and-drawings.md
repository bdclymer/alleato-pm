# Find Project Documents and Drawings

[Documents](/documents) and [drawings](/drawings) are project-specific. Select
the right project first, then open the document or drawing tool for that
project.

## Find Documents

1. Select the project in the header.
2. Open [Documents](/documents).
3. Search, filter, or sort the document list.
4. Open the document record to review details or download files when available.

## Find Drawings

1. Select the project in the header.
2. Open [Drawings](/drawings).
3. Search by drawing number, title, area, discipline, or status.
4. Open a drawing to review the current revision, related items, and viewer
   tools.

## Drawing Sets and Revisions

Use **Drawing Sets** when you need drawings grouped by issue set. Use the
**Revisions Report** when you need to understand what changed across drawing
revisions.

## If Something Looks Missing

- Confirm you selected the right project.
- Check whether the item is in documents, drawings, specifications, or another
  project tool.
- Ask [Ask Alleato](/ai-assistant) to help find the item by name, number,
  vendor, or topic.
