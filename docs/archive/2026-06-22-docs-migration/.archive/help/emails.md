# Emails

The Emails tool collects project email correspondence in one place. Incoming emails sync from Microsoft 365 and are automatically assigned to projects when the subject, body, or sender matches a project.

## Open Emails

1. Select the project.
2. Open **Emails** from the sidebar.
3. The inbox shows every email assigned to the project.

## View an Email

1. Click an email in the list.
2. The detail panel shows the sender, recipients, subject, body, and attachments.
3. Attachments can be downloaded individually or saved to project documents.

## Compose

1. Select **Compose**.
2. Enter recipients, subject, and body.
3. Attach files from your computer or from project documents.
4. Send the email.

Sent emails are stored on the project record automatically.

## Auto-Assignment

Emails are auto-assigned to projects based on:

- Project name or job number in the subject.
- Project participants on the To, CC, or From lines.
- Past assignments of the same thread.

If an email is mis-assigned, use **Reassign** to move it to the correct project.

## Attachment Search

Attachment text content is extracted and indexed. Search inside attachments using the email search bar to find specific terms.
