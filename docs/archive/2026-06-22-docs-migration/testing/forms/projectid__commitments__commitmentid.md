# Commitments Form

- Page: [/[projectId]/commitments/[commitmentId]](https://projects.alleatogroup.com/[projectId]/commitments/[commitmentId])
- Form Type: Detail with edit

| Field | Type | Description | Verification |
|---|---|---|---|
| Contract Number | Text | Provides the contract number used for unique identification and traceability. |  |
| Title | Text | Captures title required to create or update the form record. |  |
| Contract Company | Select/Dropdown | Captures contract company required to create or update the form record. |  |
| Description | Textarea | Stores narrative details for team context and audit history. |  |
| Start Date | Date (subcontract only) | Captures the start date for the record workflow. |  |
| Estimated Completion / Delivery Date | Date | Captures the estimated completion / delivery date for the record workflow. |  |
| Contract Date (Executed Date) | Date | Captures the contract date (executed date) for the record workflow. |  |
| Retention % | Number | Captures retention % required to create or update the form record. |  |
| Accounting Method | Select | Captures accounting method required to create or update the form record. |  |
| Assignee | User Select | Captures assignee required to create or update the form record. |  |
| Inclusions | Textarea | Captures inclusions required to create or update the form record. |  |
| Exclusions | Textarea | Captures exclusions required to create or update the form record. |  |
| Private | Toggle | Captures private required to create or update the form record. |  |
