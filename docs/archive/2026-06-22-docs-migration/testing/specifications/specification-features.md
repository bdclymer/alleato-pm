# Specification Features

As of **April 17, 2026**, here’s a consolidated feature list for Procore’s **Specifications** tool:

1. Enable/configure the Specifications tool at project level.  
2. Configure tool settings and advanced options.  
3. Create and manage specification sets.  
4. Upload specification PDFs.  
5. OCR-assisted extraction of spec section metadata during upload.  
6. Review and publish uploaded specs.  
7. Upload specification revisions (auto versioning/increment behavior).  
8. Auto-save/archive all revisions with latest marked current.  
9. View/open spec section PDFs on demand.  
10. Search within spec PDFs (keyword/contextual search).  
11. Search/filter the Specifications log.  
12. Bulk edit specification sections/divisions.  
13. Manually create divisions.  
14. Manually create specification sections.  
15. Mark specifications as obsolete.  
16. Move specifications between specification areas.  
17. Add/edit/delete **Specification Areas** (when enabled).  
18. Relate/link specifications to items in other tools (RFIs, Submittals, etc.).  
19. View specification references from linked records in other tools.  
20. Email specifications (web + mobile).  
21. Download specifications (including mobile options by division/individual file).  
22. Export specifications (PDF/CSV).  
23. Delete pending uploads.  
24. Delete specification revisions (where allowed).  
25. View all revisions for a spec section.  
26. View change history.  
27. View deleted specifications.  
28. Show all uploads (including uploads initiated by others, with permission).  
29. Granular permissions for fine-grained control of spec actions.  
30. Submittal Builder integration (including AI-powered submittal generation from selected spec sections).  
31. Unified mobile spec listing across iOS/Android.  
32. Modernized mobile PDF rendering/viewer experience.
