# Ask the AI Assistant for Help

Use [Ask Alleato](/ai-assistant) when you need help finding information, understanding a
workflow, or getting oriented inside the platform.

The assistant can answer questions from available project data, help
documentation, and approved knowledge sources. When an answer depends on data
that is not indexed or a workflow that is not documented yet, it should say so
plainly instead of guessing.

## Good Questions To Ask

- How do I [create a user](/docs/create-or-invite-a-user)?
- Where do I [update my profile](/docs/update-your-profile)?
- How do I [manage permissions](/docs/manage-permissions)?
- What changed on this project recently?
- What needs my attention today?

## Important Boundary

The assistant can summarize and guide, but decisions should still be verified
against the underlying record when the answer affects project, financial, or
permission-sensitive work.
