# Use Help Center and Ask Alleato Together

[Help Center](/docs) and [Ask Alleato](/ai-assistant) work together. Help
Center gives you controlled written guides. Ask Alleato can search those guides
and answer follow-up questions in plain language.

## Start With Help Center

1. Open [Docs](/docs).
2. Search for the workflow or tool you need.
3. Use categories to narrow the list.
4. Open the guide that matches your task.

## Ask Alleato For Follow-Up Help

Use [AI Strategist](/ai-assistant) when you want the assistant to explain a workflow, find a
record, summarize project data, or point you to the right place in the app.

Good follow-up questions include:

- How do I [invite a user](/docs/create-or-invite-a-user)?
- Where do I [manage permissions](/docs/manage-permissions)?
- Which project should I look at for this document?
- What should I check before changing access?

## Important Boundary

Ask Alleato can guide and retrieve approved help content. Some workflows still
require you to complete the action in the app until assistant write actions are
implemented with preview and confirmation.
