# Create or Invite a User

New users are added from [User Management](/user-management). In the current
workflow, creating a user starts by sending an invitation to their email
address.

## Steps

1. Open [User Management](/user-management).
2. Use **User Management**.
3. Select **Invite user**.
4. Enter the user's email address.
5. Add their name if you have it.
6. Choose a role when the role is known.
7. Select **Send invite**.

The invited person receives access through the invitation flow. After the
account exists, an admin can open that user record to review status, role, and
access level.

## Check Before Sending

- Use the company email address that should own the account.
- Confirm whether the person needs standard access or admin access.
- Confirm whether they need company-wide access or only project-specific
  access.

## If Something Looks Wrong

If the invitation fails or the person does not receive it, use feedback or ask
[Ask Alleato](/ai-assistant) to help identify whether the user already exists,
is inactive, or needs a resent invitation.
