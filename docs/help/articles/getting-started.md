# Getting Started with Alleato OS

Alleato OS is the construction project management platform used to run budgets, schedules, drawings, RFIs, submittals, meetings, and team operations across every project.

Use this guide to find the right area of the platform for the work you need to do.

## First Steps

1. [Update your profile](/docs/update-your-profile) so your name, photo, and contact details are correct.
2. [Navigate to a project](/docs/navigate-projects) and confirm the project context before any project work.
3. [Ask the AI assistant](/docs/ask-the-ai-assistant) for help finding records, summarizing data, or understanding a workflow.

- [Budget](/docs/budget-overview) — line items, cost codes, actual vs forecast.
- [Change Events](/docs/change-events) — potential change orders and pricing.

- [Commitments](/docs/commitments) — subcontracts and purchase orders.
