# Project Directory

The Project Directory is the roster of every person and company involved in a specific project. Use it to add subcontractors, owner contacts, design team members, and inspectors.

## Open the Project Directory

1. Select the project.
2. Open **Directory** from the sidebar.
3. The directory has tabs for **People** and **Companies** active on this project.

## Add a Person

1. Select **Add Person**.
2. Search the company directory for an existing contact, or create a new one.
3. Assign a **Project Role** (Owner Rep, Architect, Subcontractor PM, Foreman, etc.).
4. Optionally grant **Project Access** to the platform.
5. Save the entry.

## Add a Company

1. Select **Add Company**.
2. Search the company directory or create a new company.
3. Assign the company **Trade** (electrical, mechanical, GC, owner, etc.).
4. Save the entry.

## Project Access vs Directory Entry

- A directory entry is just a contact record.
- Project access grants login permissions to view the project in Alleato OS.

Granting access is optional. Many directory entries are external contacts who never log in.
