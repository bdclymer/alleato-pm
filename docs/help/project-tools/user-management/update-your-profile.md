# Update Your Profile

[Your profile](/settings/profile) shows the account details Alleato OS uses to
identify you across the platform.

## Steps

1. Open the user menu in the top-right corner.
2. Select [Profile](/settings/profile).
3. Review your name, email, title, company, phone, location, timezone, region,
   and role.
4. To update your profile image, use the profile image upload control.

## When Details Need To Change

Some profile fields are controlled by the user record rather than edited
directly on the profile page. If your name, title, role, company, or email is
wrong, ask an admin to review your record in [User Management](/user-management).

## Admin Edit Path

Admins can open [User Management](/user-management), choose the person, update
account fields, and save the record.

## Check Before Changing Access-Related Fields

Profile details can affect search, notifications, reporting, and access.
Confirm role or admin-access changes before saving them.
