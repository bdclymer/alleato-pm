# What's New

The What's New page shows recent platform updates so you can see what changed without reading internal release notes.

## Open What's New

1. Open **Updates** from the main sidebar, or
2. Open the link from the user menu when a new release banner appears.

## Categories

Each entry is tagged with one of:

- **New** — a brand-new feature.
- **Improved** — an existing feature got better.
- **Fixed** — a bug was resolved.

Entries are also grouped by area:

- AI
- Financial Tools
- Operations
- UI
- Infrastructure
- Integrations
- Security

## Filter

Use the filter to narrow by category, area, or date range. This is useful when you want to know "what changed in the last 30 days" or "what's new in financial tools this quarter."

## Subscribe

Subscribe to the changelog to get a monthly summary by email.
