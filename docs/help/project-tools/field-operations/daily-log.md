# Daily Log

The Daily Log captures what happened on the jobsite each day. Use it to record labor and equipment hours, weather, deliveries, visitors, safety notes, and observations. Daily logs are the source record for delay claims, productivity analysis, and field reporting.

## Open the Daily Log

1. Select the project.
2. Open **Daily Log** from the sidebar.
3. Use the date picker to select a day or create a new log.

## Create a Daily Log

1. Select **Create Daily Log** for the date.
2. Add entries by category:
   - **Labor** — company, trade, worker count, hours.
   - **Equipment** — type, hours used, idle hours.
   - **Weather** — conditions, temperature, precipitation, wind.
   - **Deliveries** — what arrived from whom.
   - **Visitors** — who came on site.
   - **Safety** — incidents, near-misses, toolbox talks.
   - **Notes** — general observations.
3. Attach photos or documents to any entry.
4. Save the log.

## Submit and Lock

Once a daily log is finalized:

1. Select **Submit**.
2. The log locks for further edits.
3. Project managers and superintendents can review and unlock if a correction is needed.

## Reporting

Daily logs feed weekly and monthly reports. Use **Export** to download a date-range PDF for owner reporting or claims.
