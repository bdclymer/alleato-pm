## Important caveats

- This is **static analysis**. Some "orphans" may be:
  - Called dynamically (string-built URLs, computed imports)
  - Used in tests outside `frontend/src/` (Playwright suites under `frontend/tests/`, smoke contracts under `scripts/`)
  - Imported via barrel re-exports (`index.ts` chains, `@/components/ds`, `@/hooks`)
  - Referenced only from external consumers (scripts, backend, cron)
  - Newly added and not yet wired up
- The component audit is the noisiest category. A large cluster (`components/ai-elements/` = 300 entries) comes from a third-party chat UI kit where sub-components are exported together but only a few are used at a time — many of those are **not** truly dead, just optional.
- API routes under `/api/cron`, `/api/webhooks`, and `/api/liveblocks` are called by external systems (Vercel Cron, webhook providers) and are correctly reported as having no in-repo callers — do NOT delete these without checking the scheduler config.
- Verify before deleting anything. This list is for triage, not auto-deletion.
