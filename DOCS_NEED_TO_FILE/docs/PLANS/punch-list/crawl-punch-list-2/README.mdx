# Procore Punch List Feature Crawl

Comprehensive screenshot and DOM capture of Procore's Punch List feature.

## Overview

This crawl captures the Punch List tool within Procore construction management software, including:
- Main punch list view
- Individual item PDFs
- Configuration settings
- Create/Edit forms
- My Items view
- Recycle Bin view

## Directory Structure

```
crawl-punch-list/
├── README.md                    # This file
├── PUNCH-LIST-CRAWL-STATUS.md   # Crawl execution status and statistics
├── pages/                       # Captured page data
│   ├── list/                    # Main punch list page
│   │   ├── screenshot.png
│   │   ├── dom.html
│   │   └── metadata.json
│   ├── new/                     # Create new item form
│   ├── configure_tab/           # Configuration settings
│   ├── my_items_list/           # User's assigned items
│   ├── recycle_bin_list/        # Deleted items
│   └── view_pdf_id_*/           # Individual item PDF views
└── reports/                     # Generated analysis reports
    ├── sitemap-table.md         # Markdown table of all pages
    ├── detailed-report.json     # Full metadata for all pages
    ├── link-graph.json          # Navigation structure
    └── summary.json             # Statistics summary
```

```bash

```

1. Open `reports/sitemap-table.md` for a quick overview

3. Browse `pages/` folders for screenshots and DOM captures
